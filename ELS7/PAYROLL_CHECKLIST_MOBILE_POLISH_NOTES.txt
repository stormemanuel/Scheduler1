ELS App Upgrade - Payroll rate priority, checklist popup, mobile polish

Changes in this bundle:
- Payroll now uses saved Default/City master crew rates before crew-profile position rates.
- Crew profile role rates are now treated as fallback only, preventing old client-facing rates such as $450 GAV from overriding current crew-pay defaults.
- Payroll rate source labels now identify default crew rates vs crew fallback rates.
- Event checklist is now an in-app popup/modal with checkboxes, check all, clear, copy, and export.
- The old checklist display mode was removed from Change view; checklist is now an action button.
- ELS logo is larger and more prominent.
- Mobile layout improved: Events stacks into one column, toolbars wrap, cards fit small screens, and the header/nav are more phone-friendly.

Deployment:
- package.json is at the repo root.
- Vercel Root Directory should stay blank.
- package-lock.json is intentionally not included so Vercel can perform a clean npm install.
