VS import parser + schedule conflict rule fix

This build keeps the working import flow and adds/strengthens these event fixes:

1. Same-day overlap rule
- Manual Add Crew blocks assigning the same crew member to overlapping calls on the same date.
- PDF import skips assignment rows that would double-book the same crew member at overlapping times on the same date.
- The check runs by calendar date, not just within the current event.

2. VS EXCHANGE PDF improvements
- Repairs split rows where the weekday appears first, worker details appear next, and the actual month/date/year is continued later in the row or on the next line.
- Prevents fake dates such as 2026-04-20 from being created by broken date fragments.
- Keeps OPTOS Booth 923 as its own booth/area.
- Preserves rows that continue across page breaks.
- Fixes split roles so Lighting Assist does not become plain Assist.
- Keeps LED Assist / Lighting Assist split-row wording readable and rate-compatible.

4. Existing features preserved
