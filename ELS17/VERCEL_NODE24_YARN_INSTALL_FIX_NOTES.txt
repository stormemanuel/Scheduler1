VERCEL NODE 24 YARN INSTALL FIX

This package sets Node.js to 24.x because Vercel discontinued Node 18.x for this project.
It avoids the npm installer error by using Yarn Classic through Corepack.

Vercel settings used:
- engines.node: 24.x
- packageManager: yarn@1.22.22
- installCommand: corepack enable && corepack prepare yarn@1.22.22 --activate && yarn install --network-timeout 600000 --ignore-optional
- buildCommand: yarn build

After deploy, run Supabase SQL if not already run:
- supabase/fix_crew_pay_rates.sql

Only run contractor_tax_profiles_encrypted.sql when ready to store contractor TIN/address data.
