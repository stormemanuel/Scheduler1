This package is flattened for Vercel/GitHub deployment.

Deploy instructions:
1. Unzip this package.
2. Upload/commit the CONTENTS of this folder to the root of your GitHub repo.

4. Do not upload a wrapper folder like ELS44_CLEAN_UNDER100_2026-05-20 above package.json.
