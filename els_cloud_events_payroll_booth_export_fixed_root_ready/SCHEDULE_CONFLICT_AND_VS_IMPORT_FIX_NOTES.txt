ELS Events Upgrade - Schedule Conflict + VS Import Fix

This patch keeps the existing import feature and adds three protections:

1. Same-day/same-time conflict guard
- Manual Add Crew now blocks assigning a crew member to overlapping sub-calls on the same calendar date, even across different events.
- PDF import now filters matched crew assignments through the same overlap rule.
- Import success messages report skipped schedule conflicts when a PDF tries to double-book someone.

2. VS EXCHANGE parser correction
- Fixed the split weekday/date parser bug that could turn "April 28, 2026" into fake dates like "2026-04-20".
- Added range validation so imported rows outside the event date span are skipped instead of creating fake labor days.
- Improved page-break/queued booth handling so OPTOS Booth 923 remains a booth/area and does not get swallowed by the previous Luxottica section.

3. Existing features preserved
- PDF import remains in place.
- Crew matching remains phone-first/fuzzy-name based.
- Day view / booth view remains in place.
- Inline edit behavior remains in place.
