ELS Cloud App Upgrade
Payroll Rate Fix + iMessage Schedules + Checklist + ELS Logo Colors

What changed:
1. Payroll rate matching was tightened so aliases like BO Tech, BO, Breakout Tech, GAV, AVT, LED Stagehand, CF AVT, and Client Facing AV Tech resolve to the correct crew/master-rate family.
2. Payroll now prefers a crew-specific saved role rate when it exists, then uses the selected event rate city, then Default master rates, then a built-in fallback when needed.
3. Events now include view options for iMessage schedules and checklist views in the existing Change view selector.
4. iMessage schedule export creates one copy-ready schedule per assigned tech.
5. Event editing now has message detail fields: meet-up location, default hourly message rate, crew lead name, crew lead phone, coordinator name, and coordinator phone.
6. Checklist export creates a copy-ready checkbox list in the same order the crew appears in the event schedule.
7. The UI palette now uses the ELS logo colors: dark teal / near black with gold accents. The uploaded logo is included in the app header as public/els-logo.png.

Install note:
- package.json is at repo root, so Vercel Root Directory should remain blank.
- No package-lock.json is included, so Vercel can do a clean npm install.
