ELS47 - Feedback Survey Question Update

Use the full ZIP for a clean full project under 100 files.
Use the CHANGED_ONLY ZIP if you already deployed ELS46 and only need this update.

Changes:
- Project Manager / Overall Event survey now uses the requested 5-star questions:
  Overall intent to recommend, overall planning experience, response time,
  offerings matched needs, billing experience, overall on-site experience,
  staff competence and ability, timeliness, and event success.
- Booth / Area Manager surveys are simplified and focused on on-site experience,
  staff competence, overall experience comments, testimonial permission,
  and per-tech ratings/notes.
- Wording is more like a quick Google Forms style survey and less like a test.
- HTML survey still supports fillable fields, copy completed response, and print/save PDF.

No Supabase SQL is required for this update.
