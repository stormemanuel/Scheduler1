Worker Notes Upgrade

What changed:
- Added assignment_notes storage support for professional show/person notes.
- Added API routes:
  - GET/POST /api/assignment-notes
  - DELETE /api/assignment-notes/[id]
- Added worker note dropdowns inside each assigned crew row on Events.
- Notes support multiple preset selections, optional custom text, and visibility tags.
- Added professional labels such as Late arrival, Strong work performance, Exceptional performance, Recognized by client, Requested for future work, and Do not reassign to this client/show type without approval.
- Added a Worker notes event view with export.
- Crew list HTML exports now include a Notes column.

Run once in Supabase SQL Editor if the table does not exist:
  supabase/assignment_notes_migration.sql

Do not run SQL in the Table Editor. The Table Editor is only for viewing/editing rows after the table exists.
