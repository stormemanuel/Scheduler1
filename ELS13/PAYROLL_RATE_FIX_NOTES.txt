Payroll Rate Fix Notes

What changed:
- Payroll now uses Default/city master crew-pay rates before any crew-profile role rate.
- Old crew profile rates are now only fallback values when no master rate exists.
- Event estimated payout now uses the same master-rate day/half-day logic and counts actual assigned crew when assignments exist.
- This prevents old client-facing rates like $450 General AV or $500 LED Assist from overriding crew-pay defaults.
