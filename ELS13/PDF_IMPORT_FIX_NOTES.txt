PDF import parser update

This build expands the Events PDF importer for the real ELS labor call list variations uploaded and tested:
- ADA26 Crew List.pdf
- AAPA Crew List.pdf
- KIDNEY26.pdf
- VS EXCHANGE 26 N.pdf

Parser fixes included:
- Repairs PDF glyph/black-box extraction artifacts before parsing.
- Repairs corrupted times where PDF extraction changes 8:00 AM into 8 00 AM.
- Supports rows collapsed together on one extracted line.
- Supports numeric dates like 06/03 and month dates like May 15, 2026 / Apr 29, 2026.
- Supports weekday and split weekday date rows used in VS EXCHANGE.
- Supports pipe-separated and hyphen-separated time ranges.
- Supports PO-column rows and booth/section headers.
- Skips On call / Available note rows.
- Preserves fuzzy crew matching and phone-first crew assignment.
